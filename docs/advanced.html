Advanced Usage

Using the Store

podlovePlayer(selector, config).then(function (store) {
    store.dispatch({
        type: TYPE,
        payload: PAYLOAD
    })
});

Every player interaction can be triggered via the redux store. Accessing the players store enables the full control of the player while running. The color picker is a good example.
